## ⚡ Быстрый старт (5 минут)

### 1️⃣ Активируйте виртуальное окружение

**Windows:**
```bash
.venv\Scripts\activate
```

**Linux/macOS:**
```bash
source .venv/bin/activate
```

### 2️⃣ Инициализируйте приложение

```bash
python setup.py
```

Этот скрипт:
- ✅ Применит все миграции БД
- ✅ Создаст администратора
- ✅ Опционально создаст тестовые данные

### 3️⃣ Запустите сервер

```bash
python manage.py runserver
```

### 4️⃣ Откройте в браузере

```
http://localhost:8000
```

## 🛠️ Полезные команды

### Создать больше тестовых пользователей
```bash
python manage.py create_test_data --users 20
```

### Создать суперпользователя (администратора)
```bash
python manage.py createsuperuser
```

### Django shell для экспериментов
```bash
python manage.py shell
```

### Запустить тесты
```bash
python manage.py test
```

### Собрать статические файлы (для production)
```bash
python manage.py collectstatic
```

## 💡 Советы

### Разработка
- 📌 Используйте Django shell для тестирования: `python manage.py shell`
- 🐛 Используйте breakpoint() для отладки
- 📊 Проверяйте БД запросы в консоли
- 📝 Пишите тесты для своего кода

### Хоститинг
- 🌐 Используйте HTTPS в production
- 🔐 Меняйте SECRET_KEY на production
- 📧 Настраивайте реальный email сервис
- 💾 Создавайте регулярные бэкапы БД

### Масштабирование
- 🚀 Переходите на PostgreSQL при большом количестве пользователей
- ⚡ Добавляйте кеширование (Redis)
- 🔄 Используйте Celery для async задач
- 📊 Мониторьте производительность
